📖 TAILWIND CLASSES YANG SERING DIPAKAI

LAYOUT:
  container mx-auto         → Container centered
  px-4 lg:px-8             → Responsive padding
  grid grid-cols-2         → Grid 2 columns
  flex items-center        → Flexbox centered
  space-x-4                → Horizontal spacing

SIZING:
  w-80                     → Width 320px (20rem)
  h-48                     → Height 192px (12rem)
  text-4xl lg:text-5xl     → Responsive text size

COLORS:
  bg-white                 → White background
  text-gray-800            → Dark gray text
  bg-gradient-to-br        → Gradient bottom-right

EFFECTS:
  rounded-2xl              → Large border radius
  shadow-lg                → Large shadow
  hover:shadow-2xl         → Shadow on hover
  transition transform     → Enable transitions
  hover:scale-105          → Scale on hover
  hover:-translate-y-2     → Move up on hover

🐛 TROUBLESHOOTING

MASALAH: Tailwind classes tidak bekerja
SOLUSI:
   - Cek koneksi internet (Tailwind CDN)
   - Refresh browser (Ctrl + F5)
   - Cek console (F12) untuk errors

MASALAH: Warna custom tidak muncul
SOLUSI:
   - Pastikan tailwind.config ada di <script>
   - Cek nama class sesuai config
   - Gunakan bg-adab-purple bukan bg-adab-purple-light

MASALAH: Hover effects tidak smooth
SOLUSI:
   - Tambahkan class "transition"
   - Format: transition transform hover:scale-105

MASALAH: Layout broken di mobile
SOLUSI:
   - Gunakan responsive classes (md:, lg:)
   - Test di DevTools (F12 → Toggle Device)

💡 TIPS TAILWIND

1. ARBITRARY VALUES (Custom values on-the-fly)
   <div class="w-[350px]">...</div>
   <div class="top-[13px]">...</div>
   <div class="text-[#667eea]">...</div>

2. GROUP HOVER (Parent hover affects child)
   <div class="group">
       <div class="group-hover:scale-110">...</div>
   </div>

3. SPACE UTILITIES (Spacing between children)
   <div class="space-y-4">...</div>  <!-- Vertical -->
   <div class="space-x-6">...</div>  <!-- Horizontal -->

4. GRADIENT DIRECTIONS
   bg-gradient-to-r    → Right
   bg-gradient-to-br   → Bottom-right
   bg-gradient-to-t    → Top

5. OPACITY UTILITIES
   bg-white/90         → White with 90% opacity
   text-gray-600/50    → Gray with 50% opacity
